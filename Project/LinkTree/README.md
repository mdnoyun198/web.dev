# Link Tree

A clean and responsive Link Tree style profile page built with HTML, CSS, and JavaScript. It presents personal social links, profile details, and a mobile-friendly layout in one simple landing page.

## Features

- Responsive layout for desktop, tablet, and mobile devices
- Profile section with avatar, bio, and quick stats
- Social link cards with icons and hover effects
- Lightweight structure using only HTML, CSS, and JavaScript

## Project Structure

```text
LinkTree/
├── index.html
├── style.css
├── script.js
└── img/
```

## How to Use

1. Clone or download this project.
2. Open `index.html` in your browser.
3. Update the profile details and social links to match your own information.
